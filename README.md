# Vet Clinic API

Микросервис для ветеринарной клиники.  
Хранение и обновление информации о собаках.

## Возможности:
- Получить список всех собак
- Получить собаку по ID
- Добавить собаку
- Обновить данные
- Удалить собаку

## Запуск:
pip install -r requirements.txt
uvicorn main:app --reload

## Swagger:
http://localhost:8000/docs
